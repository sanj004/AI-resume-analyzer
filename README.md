#AI Resume Analyzer & Rewriter
AI-powered tool to analyze, score and rewrite resumes using Groq AI API (LLaMA3)

#Live demo
Try it here : https://ai-resume-analyzermeh.streamlit.app/

# Features
-  Parse PDF resumes automatically
-  Score resume vs job description by keyword match
-  AI-powered analysis with strengths & weaknesses
-  Automatically rewrite resume to improve match score
-  Download the rewritten resume

#SetUp Instructions

1. Clone the repo
bash
   git clone https://github.com/sanj004/AI-resume-analyzer.git
   cd AI-resume-analyzer


2. Create virtual environment
bash
   python -m venv venv
   venv\Scripts\activate


3. Install dependencies
bash
   pip install -r requirements.txt


4. Add your API key to a `.env` file
